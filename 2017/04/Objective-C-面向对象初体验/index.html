面向对象简介

语言的面向对象都是换汤不换药 来来去去那几个概念

  • OOP（Object Oriented Programming) 基本概念
  • 对象 （男人，女人，程序员）
  • 抽象 - 类（人）

类和对象的关系

  • 创建类(人)、得到对象（程序员）
  • 成员变量： 内部使用，比如 大脑 眼睛 嘴
  • 属性： 给外部使用 比如 职业
  • 类的实例为对象
