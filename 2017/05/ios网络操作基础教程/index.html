  • NSURLRequest: 客户端发起的网络请求内容

提示我们已经阻止了明文http请求，因为他是不安全的，可以通过Info.plist配置解除请求。
在项目根目录里打开Info.plist， 添加 App Transport Security Settings 并在App Transport Security Settings下添加Allow Arbitrary Loads值为YES。
